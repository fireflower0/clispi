# color

## Description

Blink RGB Color LED

## Circuit diagram

Wire as follows

```
                           RGB Color LED
GPIO17 ----- R (1kΩ) ----- Red
GND    -------------------- GND
GPIO27 ----- R (1kΩ) ----- Green
GPIO22 ----- R (1kΩ) ----- Blue
```

## Usage

Run roswell script

```
ros roswell/color.ros
```
